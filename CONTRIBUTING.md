# Exosphere Command Sender for JS Developer Guidelines

## Install

* `npm i`
* add `./bin/` and `./node_modules/.bin` to your PATH


## Testing

- run all tests: `bin/spec`
- run linter: `bin/lint`
- run feature specs: `features`


## Update

```
$ update
```


## Deploy a new version

```
publish <patch|minor|major>
```
